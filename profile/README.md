# hoelterhoff.consulting

**SAP HCM Consulting & DSGVO-Compliance**

Wir unterstützen Unternehmen bei der rechtssicheren Umsetzung der DSGVO im SAP HCM — mit Erfahrung aus über 20 Jahren SAP-HCM-Beratung und einer eigenen Tool-Suite für die Praxis.

---

## 🛡️ HCM DSGVO PowerSuite

Modulare ABAP-Werkzeuge für das Datenschutz-Management im SAP HCM:

| Modul | Zweck |
|---|---|
| **DataScope** | Analyse & Transparenz über personenbezogene Daten im HCM |
| **RuleBoard** | Zentrale Verwaltung von Löschregeln und Aufbewahrungsfristen |
| **AutoPurge** | Automatisierte, regelbasierte Löschung nach ILM-Standards |
| **CustomClean** | Bereinigung kundenspezifischer Tabellen und Felder |
| **PayrollFix** | Korrektur und Anonymisierung in Abrechnungsclustern |
| **ResetFlag** | Zurücksetzen von Sperr- und Löschkennzeichen |
